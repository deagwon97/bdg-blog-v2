# bdg.blog

bdg.blog는 온프레미스로 쿠버네티스 위에서 동작하는 블로그 서비스입니다. bdg.blog는 DNS 라우팅 기능(AWS Route 53)과 형상 관리 기능(Github)을 제외한 모든 기능을 온프레미스로 배포하고 있습니다. 개인적으로 공부한 내용을 정리하고, 적용하는 목적으로 개발 중 입니다.
